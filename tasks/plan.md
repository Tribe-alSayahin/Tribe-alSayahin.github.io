# خطة إعادة تصميم موقع قبيلة السياحين

## الهدف
إعادة تنظيم أقسام الموقع ضمن رحلة سردية مترابطة، وتغيير التصميم كلياً بأسلوب إبداعي يحافظ على هوية القبيلة والنظام البصري الحالي (ألوان النحاس/الحبر، الخطوط العربية، RTL، الوضع الفاتح/الداكن).

## الحالة الحالية
- عدد كبير من الأقسام (١٣+) دون تجميع موضوعي واضح.
- الترتيب الحالي: Hero → Jathum → Lineage → Map → Gallery → Wasm → Constellation → Archive → Poetry → News → Admin → Timeline → Supporters → Contact.
- بعض الأقسام (Timeline, Supporters, Contact) لا تستخدم المكوّن `Section` الموحّد، مما يُحدث تبايناً بصرياً.
- أرقام الأقسام غير متسقة (٠٨.٥، ٠٨.٧).
- شريط التنقل يحتوي على ١٤ رابطاً، وهو أكثر من المريح على سطح المكتب والجوال.
- أقسام الشعر والأرشيف لا تحتوي على محتوى موثق بعد، وتظهر كحالات فارغة.

## المفهوم التصميمي: "ديوان السياحين الرقمي"
تحويل الموقع إلى مخطوطة رقمية/ديوان تفاعلي يأخذ الزائر في رحلة سردية عبر:
1. **الفاتحة** (الرئيسية)
2. **الفصل ١: الأصول** — الجثوم، النسب، الأنساب
3. **الفصل ٢: الديار** — الخريطة، معرض التراث
4. **الفصل ٣: الهوية** — الوسم، الشعر
5. **الفصل ٤: التاريخ والشهادات** — الخط الزمني، الأرشيف
6. **الفصل ٥: المجتمع** — الأخبار، الداعمين، التواصل، الإدارة

كل فصل يُفصَل بفاصل فصل (`ChapterDivider`) كبير يحمل رقم الفصل والعنوان والوصف، ويستخدم نقوش السدو والزخارف النحاسية.

## التغييرات التصميمية الإبداعية

1. **مكوّن فاصل الفصل (`ChapterDivider`)**: شريحة كاملة العرض بين الفصول تحمل رقم الفصل بالخط الكوفي الكبير، عنوان الفصل، وشريط سدو.
2. **تحديث `SectionHeader`**: إضافة رقم الفصل، توسيع الرقم الشبحي، إضافة زخرفة نحاسية في الأعلى.
3. **تحديث `Section`**: دعم خاصية `chapter` و `tone` و `surface` لإنشاء تباين خفيف داخل الفصل.
4. **تبسيط شريط التنقل**: ٥-٦ عناصر فقط (الأصول، الديار، الهوية، التاريخ، المجتمع، تواصل) مع قائمة منسدلة للأقسام الفرعية على الجوال.
5. **تصميم Hero جديد**: إضافة "مدخل الديوان" مع شريط فصول سريعة وإشارة تمرير سينمائية.
6. **توحيد الأقسام المنتهية**: تحويل `Timeline` و `Supporters` و `Contact` لتستخدم `Section` الموحّد.
7. **إعادة تصميم `AdminSection`**: جعله بطاقة بوابة صغيرة في نهاية المجتمع بدلاً من قسم كامل.
8. **تحديث Footer**: إضافة روابط الفصول السريعة، شعار كبير، وخلفية نقوش سدو.
9. **تأثيرات بصرية جديدة**:
   - غبار ذهبي متحرك خفيف في الخلفية.
   - تأثير "ورق المخطوطة" مع نسيج خفيف.
   - توهج نحاسي متدرج يتغير مع الفصل.
   - زوايا نحاسية مزخرفة على البطاقات.

## التحقق النهائي
- `npm run lint` بدون أخطاء.
- `npm run build:pages` بنجاح.
- `npm run test` يمر.
- فحص إمكانية الوصول (axe) لا يُظهر أخطاء critical/serious جديدة.
- معاينة محلية `npm run dev` ومراجعة التصميم والتنقل.

## المخاطر
- تغيير الترتيب يُبقي المعرفات `id` ثابتة، فلا تتأثر الروابط المباشرة.
- المكونات الثقيلة (الخريطة، شجرة النسب) تحتاج إلى اختبار بعد إعادة الترتيب.
- الوضع الفاتح يحتاج إلى اختبار خاص بعد أي إضافة للألوان.
