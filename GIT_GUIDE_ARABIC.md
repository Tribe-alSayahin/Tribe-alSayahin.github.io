# خطة إنشاء دليل Git باللغة العربية

## الهدف
إنشاء دليل شامل باللغة العربية يشرح كيفية استخدام Git للمرحلة، الالتزام، والدفع إلى المستودع البعيد، مع أفضل الممارسات وحل النزاعات.

## المحتوى المقترح للدليل
1. مقدمة
    - ما هو Git ولماذا نستخدمه؟
2. إعداد المستودع (اختياري للمبتدئين)
    - `git clone <url>`
    - `git init`
3. المرحلية (Staging)
    - `git add <file>` لتحديد ملف معين
    - `git add .` لجميع الملفات في المجلد الحالي
    - `git add -p` للمرحلة التفاعلية
4. الالتزام (Commit)
    - `git commit -m "رسالة commitment وصفة"`
    - أفضل الممارسات لرسائل الالتزام:
      * استخدم زمن المضارع (مثل: "إضافة" وليس "أضافت")
      * ابدأ بحرف كبير
      * لا تنهي الرسالة بنقطة
      * احصر السطر الأول في 50 حرفًا كحد أقصى
      * استخدم الجسم لشرح التفاصيل إذا لزم الأمر (بعد سطر فارغ)
    - مثال: 
      ```
      git commit -m "إضافة قسم الأخبار والفعاليات مع البيانات الثابتة"
      ```
5. الدفع (Push)
    - `git push origin <branch>` لتحديث الفرع البعيد
    - للدفع لأول مرة وتعيين الفرع العلوي: `git push -u origin <branch>`
6. أفضل الممارسات
    - الالتزام المتكرر والمنطقي (كل Commit يمثل تغييرًا واحدًا منطقيًا)
    - استخدم ملف .gitignore لتجنب تتبع الملفات المؤقتة والملفات المبنية
    - اعمل على فرع منفصل لكل ميزة أو إصلاح
    - قبل بدء العمل، أحدث فرعك من الفرع الرئيسي: `git pull origin main`
    - راجع 변경اتك قبل الالتزام باستخدام `git diff`
7. حل النزاعات (Merge Conflicts)
    - متى يحدث النزاع؟ عند سحب التحديثات (`git pull`) أو دمج الفرع (`git merge`) وتغيير نفس الجزء من الملف في cả الفرعين.
    - خطوات الحل:
      1. عندما يحدث نزاع، سيقوم Git بتحديد الملفات المتضاربة.
      2. افتح كل ملف متضارب وابحث عن العلامات: `<<<<<<<`, `=======`, `>>>>>>>`.
      3. اختر التغييرات التي تريد الاحتفاظ بها أو دمجها يدويًا.
      4. بعد حل النزاع في الملف، قم بالمرحلة: `git add <file>`
      5. بعد حل جميع النزافات، commit: `git commit`
      6. أخيرًا، ادفع التحديثات: `git push`
    - نصيحة: استخدم أدوات مرئية لحل النزاعات المتوفرة في بيئات التطوير مثل VS Code.
8. خلاصة
    - تشجيع الالتزام بالممارسات الجيدة لتسهيل التعاون وتتبع التاريخ.
